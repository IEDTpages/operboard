# Исправление загрузки ProFinance

Версия v3 ждёт фактического заполнения `table_history`, читает скрытые ячейки через
`textContent`, определяет колонки даты и Close по заголовкам и поддерживает даты с
двузначным годом и русскими названиями месяцев.

При повторной ошибке лог теперь содержит `sample=[...]` с первыми строками таблицы.
Этот фрагмент позволяет точно адаптировать парсер, если ProFinance снова изменит DOM.
