# Оперборд — HTML-версия отчёта Power BI

В комплекте сохранены 11 страниц исходного отчёта: сводный дашборд, ключевая ставка, ИПЦ, курс USD/RUB, Brent/Urals, TTF, CIF ARA, LME, пшеница, подсолнечное масло и Ормузский пролив.

## Быстрый запуск с автоматическим обновлением

### Windows
1. Установите Python 3.10 или новее.
2. Запустите `start_windows.bat`.
3. Откроется адрес `http://127.0.0.1:8765/index.html`.

### Linux / macOS
```bash
chmod +x start_linux_macos.sh
./start_linux_macos.sh
```

Сервер обновляет данные при запуске и далее каждый час. Интервал можно изменить:
```bash
python server.py --refresh-minutes 30
```

## Автономный режим

`index.html` можно открыть напрямую двойным щелчком. Фильтры, навигация, масштабирование графиков и экспорт CSV будут работать, но данные останутся снимком из исходного PBIX. Для загрузки новых данных нужен запуск `server.py` — браузеры не разрешают надёжно обращаться к внешним сайтам из локального HTML из-за CORS.

## Источники
- ЦБ РФ: ключевая ставка и официальный курс USD/RUB;
- ProFinance: Brent, Urals, TTF, CIF ARA и LME;
- Московская биржа ISS: WHFOB и SOEXP;
- Bizon / данные Росстата: недельный ИПЦ;
- ArcGIS / IMF PortWatch: проход судов через Ормузский пролив.

## Надёжность обновления

Если один источник временно недоступен или изменил структуру страницы, приложение сохраняет последний успешно полученный ряд, а статус источника записывает ошибку в `data/current.json`. Наиболее чувствительны к изменению HTML-разметки страницы ProFinance; их селектор находится в `refresh_data.py` в функции `fetch_profinance`.

## Состав
- `index.html` — автономный интерактивный HTML;
- `server.py` — локальный веб-сервер и расписание обновления;
- `refresh_data.py` — загрузка и обработка веб-данных;
- `data/current.json` — актуальный кэш;
- `data/snapshot.json` — исходный снимок из PBIX.
